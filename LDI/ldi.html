LDI — Ligase Dependency Index
AUC-based LDI (primary method)
For each KO condition, a 4-parameter logistic curve is generated using the entered DC50 and Dmax. The area under that curve is integrated numerically in log10 concentration space over [Cmin, Cmax].

LDIAUC = (AUCVHL KO − AUCCRBN KO) / (AUCVHL KO + AUCCRBN KO)

This captures both potency (DC50) and maximal degradation (Dmax) in a single metric.

Interpretation
+1.0 = fully CRBN-dependent (CRBN KO abolishes activity)
0.0 = dual / non-selective (neither KO matters)
−1.0 = fully VHL-dependent (VHL KO abolishes activity)

Traditional DC50 LDI
Also shown for comparison: uses only DC50 values and ignores Dmax.
LDIDC50 = (DC50CRBN KO − DC50VHL KO) / (DC50CRBN KO + DC50VHL KO)

Hill slope
The Hill coefficient controls the steepness of the sigmoidal curve. Typical value for PROTACs: 1–2. Higher values produce sharper transitions. Changing the Hill slope affects AUC (and therefore LDI) but not DC50-based LDI.
