The Hub · v1.0.27
v1.0.27 — 12 Jun 2026
● Echo Plots Flagged checkbox: now scoped to the assays currently plotted on the X/Y axes — a compound flagged in a different assay no longer disappears
● Echo Plate "Colour by Flag status": rebuilt the lookup so each plate uses its OWN assay's Flag (was overwriting per Sample_ID across assays, hence "most plates red"). Plate → assay resolved from matConfigs prefix; added a proper flag legend below each plate
● Echo Curves canvas: now renders square (was stretched / shallow rectangle filling the wrap)

v1.0.26 — 12 Jun 2026
● Echo 4PL fitter rolled back to the v0.9.96 behaviour (max-|Δp| < 1e-8 convergence + 200 iters; λ ×3/÷3; safety bounds restored on Free params). Audit blocks updated. The Prism-matching push made curves look worse, so this reverts the v1.0.16/20/21 changes
● Echo Curve PDF export: when there are multiple assay/groups, clicking Generate curve PDFs now opens a picker with checkboxes so you can choose which (assay, group) combinations to export. Single-group runs go straight through. Filenames include the assay type
● Echo multi-assay Plots: first view of the Plots tab now defaults to assay-vs-assay LogIC50 (X = first assay's LogIC50_M, Y = second assay's LogIC50_M) so the cross-assay comparison is obvious
● Echo Plots Flagged checkbox: now works in multi-assay mode (was filtering on undefined r.Flag on pivoted rows; now checks any <assay>::Flag === 'Yes')
● Echo Plate tab "Colour by Flag status": now uses the compound-level Flag from the 4PL fit (red = flagged, green = passed, neutral = no result); previously ran an SD-outlier check whose default green looked identical to the control-well green
● Echo Curves » Compare: added a Sample-ID search bar above the compound list so you can filter to the curve you need

v1.0.25 — 12 Jun 2026
● LabMate: removed orphan "Compound Lookup" (cheminfo) feature — scaffolded CSS / JS / Marvin-editor modal were never wired to any HTML or call site. Sidebar labels, sub-tab routing, quick-notes entries and _NEVER_COLLAPSE list all cleaned up

v1.0.24 — 12 Jun 2026
● Cross-app visual polish pass: every app gained --shadow-xs/sm/md/lg/xl CSS vars with theme-aware fallbacks (shadows now read on dark mode); global ::placeholder rule so form placeholders match var(--text3); theme-slider thumb uses var(--surface) instead of hardcoded white; Hub home cards gain a subtle background tint on hover
● Echo: dropzone hover uses theme-aware --accent2-soft; Setup modal backdrop adapts to theme; modal card gains a border
● LabMate: --error & --error-soft variables (visible on both themes); toast uses theme vars instead of hardcoded #222/#fff
● Lab Designer: gel-canvas well fills are now theme-aware — MW / DMSO / sample wells visible on dark mode (were hardcoded light greys)
● Degradation Explorer & Protein Tools: inline onmouseover="this.style..." handlers moved to CSS :hover rules

v1.0.23 — 12 Jun 2026
● Echo Setup modal no longer blocks Overview & Gradient Planner. Close (×) and Close buttons now always available; switching to Overview or Gradient Planner auto-dismisses the modal. The modal is still auto-opened on first load (and via the ⚙ Setup button) for the Data Analysis flow

v1.0.22 — 11 Jun 2026
