Guide

Properties tab

  • Paste any amino acid sequence (1-letter code or FASTA format) into the text area.
  • Results update instantly: MW, pI, ε280, GRAVY, instability index, and AA composition.
  • Enter an A280 absorbance value to calculate protein concentration via Beer-Lambert.
  • The Hydrophobicity Profile (Kyte-Doolittle, window 9) and Charge vs pH charts update automatically. Hover over the hydrophobicity chart to see residue details; drag to zoom, double-click to reset.

Cleavage tab

  • Select a protease (Trypsin, LysC, AspN, GluC, or CNBr).
  • Results show all predicted peptides with position, length, and mass.
  • Useful for planning mass-spec experiments or gel analysis.

Structure tab

  • Enter a gene name (e.g. KRAS), protein name, UniProt accession, or 4-character PDB ID (e.g. 4OBE).
  • Toggle between AlphaFold (predicted structures) and PDB (experimental structures). For PDB, non-ID queries show a list of results to choose from.
  • Use the sidebar controls to change representation (cartoon, tube, ball+stick, surface, spacefill) and colouring (by chain, secondary structure, element, hydrophobicity, B-factor/pLDDT).
  • Export a PNG snapshot or download the raw PDB/CIF file.
  • AlphaFold structures include a PAE map and pLDDT colour key in the sidebar.

Target Intel tab

  • Enter a human gene name (e.g. BRAF) or UniProt accession.
  • Retrieves: UniProt annotation (function, diseases), ChEMBL drug mechanisms, Open Targets disease associations, PDB structure IDs, STRING network link.
  • Links open directly to the relevant database pages.
