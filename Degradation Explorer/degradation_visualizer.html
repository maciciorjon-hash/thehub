Load Data

Drag and drop an Excel (.xlsx) file onto the drop zone, or click to browse. The app automatically detects target columns and assay metadata. Use Replace all to start fresh or Merge / append to combine datasets.

Table

Displays all loaded compounds with their DC50 and Dmax values per target. Use the search box and sliders to filter. Export the table as PNG (white/black/transparent background) or as CSV for downstream analysis.

DC50 vs Dmax

Scatter plot of degradation potency (DC50, X axis, log scale) vs maximum degradation (Dmax, Y axis). Points are coloured by target. Use the quality pills to highlight compounds meeting "Good" (Dmax ≥ 85%, DC50 ≤ 50 nM), "Moderate", or "Weak" criteria. Toggle labels and error bars as needed. Export PNG for presentations.
