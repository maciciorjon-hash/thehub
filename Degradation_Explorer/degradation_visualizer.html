Load Data

Drag and drop an Excel (.xlsx) file onto the drop zone, or click to browse. The app auto-detects target protein columns and assay metadata (cell line, time point, assay type). Set quality thresholds (Dmax ≥ and DC50 ≤) to define what counts as "Excellent".

Table

All loaded compounds with DC50 (nM, 1 dp) and Dmax (%, integer) per target. Search by name, filter with sliders, or apply quality pills. Click any row to expand a detail card. Export as PNG (white / black / transparent) or CSV.

DC50 vs Dmax

Scatter plot with DC50 on the log X axis and Dmax on the Y axis. Points coloured by target. Use quality pills, sliders, and the search box to focus on subsets. Toggle Zones to overlay Hit / Weak / Selective / Inactive quadrants with adjustable thresholds. Toggle Error bars to show SD. Hover a point for a detailed tooltip (shows structure if SMILES are loaded). Export PNG.

Properties

Upload a SMILES file (CSV or Excel with Compound and SMILES columns) to compute molecular properties. RDKit runs in-browser — no data leaves your machine. The table shows 2D structure, MW, logP, HBA, HBD, TPSA, and RotBonds alongside the assay DC50 / Dmax from the loaded dataset. Structures also appear in scatter hover tooltips.
