Guide

Selecting wells

  • Click a well to select it. Ctrl+click to toggle individual wells. Shift+click to extend selection as a rectangular range.
  • Click & drag anywhere on the plate to rubber-band select a region.
  • Click row letters or column numbers on the canvas edges to select entire rows or columns.
  • Use Select All in the selection toolbar, or Ctrl+A, to select all wells.

Keyboard shortcuts

  • Arrow keys — move the active well one step; the white inner ring marks it.
  • Shift+Arrow — extend the selection rectangle (like Excel).
  • Delete / Backspace — clear the type from all selected wells.
  • Enter — move active well down; Shift+Enter moves up.
  • Tab — move active well right; Shift+Tab moves left.
  • Ctrl+A — select all wells.
  • Escape — deselect all wells.

Assigning well types

  • Select wells, then click a well type in the Selection toolbar that floats below the canvas.
  • The Well Types panel (drag by its handle) lists all types — click to activate, then drag across wells to paint. Rename a type by editing its name inline.
  • Add custom types with the + Custom button in the Well Types panel.
  • Right-click a well (or selection) to clear its type.

Labels and dilutions

  • With wells selected, type in the Label field to annotate them.
  • For a pure row or column selection of ≥ 2 wells, the Dilution series tool applies concentration labels automatically.

Export

  • Click Export PNG to download a 600 DPI image with text auto-fitting inside wells.
  • The export always uses a white background regardless of theme.
