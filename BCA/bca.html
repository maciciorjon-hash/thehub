Import — Plate Reads

Works with any plate reader: open its own export in Excel, copy an 8×12 range of absorbance values (no headers), click cell A1 on the grid at the top of the Workflow tab, then paste (Ctrl/Cmd+V). You can also type values in by hand. Click "Use this data" once it's filled in.

Importing a new plate resets any standards or samples you've already painted — finish one plate's analysis (or export it) before loading the next.

⚗ Load test data fills in a synthetic demo plate (standards, samples, and one deliberately bad replicate read) so you can explore the app without a real file.

Standards — BSA Curve

Painting standard points

  • Pick a role by dragging: drag across a technical-duplicate pair of wells to create one standard point, then type its known BSA concentration (µg/µL) in the list on the right.
  • Include a blank (0 µg/µL) — it anchors the curve.
  • Click an already-painted well once (a plain click, not a drag) to exclude/include it from that point's average — use this for a detector overflow, an obvious pipetting outlier, or any read you don't trust. Excluded wells stay visibly assigned (dimmed with a ✕) so the record of what happened isn't lost.

Linear vs. quadratic fit

BCA is genuinely nonlinear across a wide concentration range — a straight line biases interpolated concentrations, especially near the top of the curve. Use Quadratic (needs ≥3 points) unless your standards span a narrow, clearly linear range.

Sample prep — Concentration, Dilution & Normalize

The Setup tab paints samples; the Results tab sets the dilution target and shows one combined table — everything about how to prepare each sample — computed from it. Paint each sample's duplicate-well pair the same way as standards, then name it (editable).

Assay-well concentration = standard curve⁻¹(mean absorbance) Final concentration = assay-well concentration × dilution factor

Dilutions are usually the same for every sample on a plate, so one Default dilution field (10× out of the box) applies to all of them — override an individual sample's own dilution field only when it was actually prepared differently. A sample using the default is shown as e.g. "10× (default)" rather than a plain "10×", so it's always clear which samples were explicitly checked.

Extrapolated flags a result outside the standards' concentration range — treat it with caution, or dilute the sample and re-run. Only 1 well means one replicate was excluded as a QC call, so the "average" is really a single read.

Normalize — dilute to target

Set one target concentration and volume in Results, applied to every sample:

Protein (µL) = target concentration × target volume ÷ final concentration Water (µL) = target volume − protein (µL)

A sample that's already more dilute than the target concentration can't be concentrated by adding water — it's flagged "too dilute" in the Results table, rather than showing a negative water volume.
